02 — Capabilities

Not just optimisation.
Portfolio intelligence.

AI-Driven Narratives

Every optimisation ships with a structured GPT-4o-mini narrative. The model receives only the math output — never raw news — and must produce exactly three bullets at temperature=0.3. No fabricated numbers. Returns null gracefully if the key is missing.

// ai_summary output
Risk: Commodities at cycle peak...
Opportunity: Domestic rate tailwind...
Action: Trim BHP 5% on strength...

ASX-Native Data Layer

Two years of adjusted daily close data for ASX200's top 20 equities by market capitalisation. Scraped via YahooQuery, stored in SQLite, and refreshed at 4:15 PM AEDT every weekday after market close. Partial-failure tolerance — one stale ticker won't break the response.

BHP CBA CSL NAB WBC ANZ WES MQG WOW RIO FMG TLS +8 more

Modern Portfolio Theory

Powered by PyPortfolioOpt 1.5.5 — Ledoit-Wolf covariance shrinkage improves stability with short histories. Objective is configurable via risk_tolerance. Per-ticker weight bounds passed directly to the convex solver.

risk_tolerance: 0.0 → min volatility
risk_tolerance: 0.7 → blended target ←
risk_tolerance: 1.0 → max Sharpe

Compliance-Ready Responses

Every response carries jurisdiction-aware disclaimers (AU 2026), structured metadata with engine version and data provenance, and a calculated_at ISO timestamp. Designed to drop straight into adviser workflow tools without additional wrapping.

SHA-256 Request Caching

Identical requests are served from a 5-minute in-memory TTL cache. The cache key is a canonical SHA-256 hash of the request — parameter order differences don't break hits. P95 latency under 200ms. Upgradeable to Redis for multi-worker deployments.

03 — Developer First

Built for engineers.
Trusted by advisers.

Frontir is API-first by design. Swagger docs live at /docs — interactive, type-safe, and runnable from the browser. Pydantic v2 validates every request so integrations fail fast, not silently.

OpenAPI 3.1 schema

Machine-readable spec for client codegen in any language — Python, TypeScript, Swift.

Structured JSON logging

structlog in production. Ship directly to Datadog, CloudWatch, or Loki without transformation.

Graceful partial failures

Tickers with insufficient history return actionable warnings, not HTTP 500. The optimisation continues with the remaining assets.

Self-hostable · MIT licensed

Deploy alongside your existing stack. No external dependency for the core engine — AI summaries are opt-in via OPENAI_API_KEY.
